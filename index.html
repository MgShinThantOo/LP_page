ButterFly Program とは?
Butterfly Program が何なのかをご紹介させて頂きます。 日本に来てシステムエンジニアとして働きたいミャンマー人若者又は株式会社Bliss Stock社に就職したい方を対象として、OJT研修プログラムです。1回のプログラムは（10） 名のみ受け入れております。
ButterFly Program 何をしなければなりませんか?
社会に入ると、必ず知っておきべきことがございます。職場が学校ではないので、雇用主が一つ一つの仕事のやり方を教える時間がなく、事前に学ぶこと、足りない知識或いはスキルを磨く必要があります。例えば、オフィス ソフトウェア ツールの使用、 プロジェクトに配置されたら、工程より作成必要な資料、与えられた作業をゴールまで迎えるようにチームとコミュニケーションを取る作戦などの努力が必要です。
ButterFly Program どれくらい時間がかかりますか? 制限は何ですか?
ButterFly Program が3ヶ月かかります。応募したい方が一つの開発言語でソフトウェア開発経験が必要です。OJTプログラム終了後、株式会社Bliss Stockに採用又は日本のＩＴ企業に採用向けなので、Butterfly Programに参加したい方が既に日本語N4レベルがあることを希望しております。このプログラムの目的が日本のＩＴ企業に働く方への教育なので、自分自身が学びたいやる気、どんな時でも乗り越えたい、頑張りたいとういうやる気が必要です。
